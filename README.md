# litert-cpp-dist
Prebuilt LiteRT Next artifacts for MihonOCR. Clean workaround for issues faced when building in the main project.

LiteRT Repository: https://github.com/google-ai-edge/litert

* Note: Copying permission declarations from [here](https://github.com/google-ai-edge/LiteRT/blob/main/litert/kotlin/src/main/AndroidManifest.xml) into an app's `AndroidManifest.xml` is required for GPU acceleration on Android 12+ (API 31+).
